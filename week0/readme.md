## 목적

  | 영역  | 활용 스택  |

  |BACKEND  | SpringBoot  |
  |DATABASE|  MYSQL + MyBatis|
  |FRONTEND| Base( HTML/CSS/JS ) -> 템플릿 제공 개별 자유|
